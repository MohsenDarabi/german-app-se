Character Base Definition – Tom

This document defines the complete and independent identity of the character Tom.
All future prompts (head, full body, walking, gestures, videos)
must strictly follow this base definition.

Character:
Name: Tom
Age: 35–37 years old
Gender: Male
Nationality: German / European

Professional adult character for a language learning application.
Represents maturity, reliability, and calm confidence.

Overall Personality:

- Confident

Never intimidating, never cold, never exaggerated.
